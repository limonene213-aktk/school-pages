Laravel や PHP の演習で利用するリポジトリと資料。

  • PHP / Laravel資料 Google Drive 授業で配布するスライドや補助資料をまとめています。

  • PHPとHTML/DB GitHub 基本的なWebアプリを作る例を段階的にまとめています。
  • PHPと通信関連のコード GitHub API連携や外部サービスと通信するコード例です。
  • PHPにおけるオブジェクト指向例 GitHub クラス設計やオブジェクト指向の考え方を練習するためのリポジトリ。
  • Laravel（test-project） GitHub 授業で扱うメインのLaravelプロジェクト。授業後に内容を最新化しています。
  • Laravel（blade-example） GitHub Blade テンプレートの基礎と MVC の流れを理解するための例。
  • Laravel（component-example） GitHub Laravel コンポーネントを学ぶための基本的な実装例です。
